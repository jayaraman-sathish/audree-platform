      Enterprise Copilot

      One chat for every business question — the user never selects an agent. The Intent Engine identifies the intent from natural language and the Orchestrator routes to the right agent automatically (PPIC, Inventory, QA, Procurement, Executive…) per the Intent–Agent Mapping. It asks back when inputs are missing, recalls session memory, computes decisions with risk & confidence from simulated SAP · WMS · LIMS data, and routes approvals per the Workflow Mapping. Every step is audited.

      Configuration Masters

      The metadata that drives the platform — the same tables as the cfg.* database schemas, and fully editable (add / edit / deactivate rows) because the platform is configuration-over-code: new intents, capabilities, agents, rules and workflows are added here, never in application code. Editing is governed: admin-only, version-controlled, publish-approved, and audited — the runtime always uses the last published version.
